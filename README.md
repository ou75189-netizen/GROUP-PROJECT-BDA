1. Project Overview

The Smart-School Information Management System is a static, multi-page web prototype that digitalises core school administration tasks. It replaces paper-based record keeping with a clean, role-based web interface where Admins, Teachers, Parents, and Students each see only what is relevant to them.

This is a static prototype, no backend, no real database. All data is mocked using JavaScript objects and persisted in the browser using local storage. The goal is to demonstrate mastery of HTML, CSS, and JavaScript fundamentals (variables, functions, conditionals, form validation, DOM manipulation).


2. Problem & Target Audience

Problem: Many public secondary schools in The Gambia rely on old, paper-based systems. This causes loss of files, slow academic reports, confusion over school fees, and weak parent–teacher communication, which hurts student progress.

Solution: A simple, mobile-friendly website for tracking attendance, grades, and fees. Records are digital, administration is faster, data is safer, and parents can check their children's performance from anywhere.

3. Target Users and roles;

|. Admin   | Manages users (add/edit/delete), views all school-wide data, configures fees structures 
|. Teacher | Records attendance, enters grades, posts announcements 
|. Parent  | Views their child's grades, attendance, and fee balance 
|. Student | Views own grades, attendance, timetable, and announcements
